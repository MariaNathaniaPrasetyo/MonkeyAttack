# 🐒 MONKEY ATTACK 
**Identifikasi Pola Kepadatan Populasi _Macaca fascicularis_ di Indonesia Menggunakan Pendekatan _Unsupervised Machine Learning_ untuk Analisis Potensi _Human-Wildlife Conflict_**

Proyek ini merupakan bagian dari Mini Project Praktikum Bioinformatika, Fakultas Biologi, Universitas Gadjah Mada.

## 👥 Tim Peneliti 
Maria Nathania Prasetyo 23/518903/BI/11301

## 📌 Latar Belakang
Konflik antara manusia dan satwa liar, khususnya Monyet Ekor Panjang (*Macaca fascicularis*), semakin sering terjadi akibat penyusutan habitat. Proyek ini memetakan titik-titik rawan konflik (hotspot) di Indonesia menggunakan *occurrence data* dari Global Biodiversity Information Facility (GBIF) yang diklasterisasi menggunakan algoritma K-Means.

## 🛠️ Teknologi yang Digunakan
* **Bahasa:** Python 3.x
* **Data Manipulation:** Pandas
* **Machine Learning:** Scikit-Learn (K-Means Clustering)
* **Visualisasi Spasial:** Folium, Matplotlib, Seaborn

## 🚀 Cara Menjalankan Proyek (Local / Codespaces)

Repositori ini sudah disiapkan agar dapat langsung dijalankan (reproducible) tanpa *error* konfigurasi.

**Langkah 1: Clone Repositori**
```bash
git clone https://github.com/USERNAME_GITHUB_KAMU/XX_MONKEY_ATTACK.git
cd XX_MONKEY_ATTACK
```

**Langkah 2: Install Dependencies**
Pastikan Python sudah terinstal, lalu jalankan perintah ini di terminal:
```bash
pip install -r requirements.txt
```

**Langkah 3: Jalankan Notebook**
Buka file `monkey_attack.ipynb` menggunakan Jupyter Notebook, VS Code, atau GitHub Codespaces, lalu jalankan *Run All Cells*. Peta interaktif akan otomatis di-*render* di bagian bawah *cell*.
